# livetest

Live testing with node.

## Dependencies
* [MongoDB](https://docs.mongodb.com/manual/installation/)
* [npm](https://www.npmjs.com/get-npm)
* [Git](https://git-scm.com/downloads) (optional)

## Install
1. Get files
  * Git clone repository
  * Or download and unzip
2. cd into directory
3. $npm install
4. $npm run build

## Run
1. $sudo mongodb
2. $npm run bots
